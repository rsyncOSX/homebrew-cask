# RsyncOSX Cask

## How do I install these formulae?

`brew install rsyncosx/cask/<formula>`

Or `brew tap rsyncosx/cask` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rsyncosx/cask"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
